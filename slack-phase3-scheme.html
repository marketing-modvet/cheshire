Clinics — the structural fix

Today: every clinic = one giant chat with vets + nurses + reception + assistants. Bad signal-to-noise. Fix: 3 channels per active branch.

The 3-channel pattern

clinic-XXX 🌐 all staff — branch news, handovers
clinic-XXX-med 🔒 vets + nurses + medical assistants — clinical
clinic-XXX-fd 🔒 reception + front-desk — bookings, CX

Optional 4th if branch manager asks: clinic-XXX-mgmt 🔒 (clinic mgr + vet lead + reception lead).

Intercommunication scheme

The structure only works if people know where to send asks and where to escalate. Three mechanisms wire it together.

1. Inbound asks → help-* channels (the "front door")

Anyone in company → help-marketing → @marketing triages
Anyone → help-it → @it triages
Anyone → help-people · -finance · -ops → dept lead triages

Triage decision: stays in help-channel (simple) OR converted to a List item / moved to relevant team-* for actual work.

2. Cross-team coordination → va-management + share-to-channel

Strategic / cross-functional → va-management all team leads see it
Specific input from another team → @-mention via "share to channel" audit trail in both

3. Brand / clinic ↔ functional teams

clinic-mv-mb-fd → billing query → help-finance
clinic-mv-mb-med → case escalation → team-clinical-vets @vets-all
clinic-mv-mb → maintenance → help-ops or team-ops-maintenance
team-clinical → product change → brand-mv → announce → clinic-mv-* channels
brand-mv → price change → team-finance-pricing + clinic-mv-*

4. Vendor ↔ internal flow

vendor-synergi → escalation → team-tech-it → if budget → va-leadership
vendor-eurovets → stock → team-ops-purchasing
vendor-ecre8 → deliverable → proj-floof-shopify or proj-pr-2025-2026

5. Project flows — pull resources temporarily

proj-pricing-2026 draws from → team-finance-pricing brand-mv / brand-bovc va-leadership
proj-udo-inventory draws from → team-ops-inventory team-tech-software vendor-udo

When project ends → archive · decision log moves to relevant team-* canvas.

The escalation ladder (one-page summary)

Day-to-day ──► clinic-* / team-* sub-channels │ Cross-team ──► help-* (request-driven) or @-mention partner team │ Strategic ──► va-management (all leads) │ Decisions ──► va-leadership (Mansoor + direct reports) │ Read-only ──► va-announcements (everyone sees outcome)
