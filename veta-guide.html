Quick Start — 3 Steps

1

Open the chat

Go to https://vetalliance.website in your browser. You'll see a token field — paste the gateway token and click Connect. Your browser remembers it from next time.

2

Start talking to Veta

Just type naturally. Ask Veta to draft documents, analyse data, research topics, create reports, summarise content, or help plan projects. No special commands needed.

3

Share files via File Hub

Go to https://vetalliance.website/files, enter the password, and drag & drop any PDF, Word, Excel, or CSV file. Then tell Veta the filename in chat — e.g. "Please analyse the file Q1-report.xlsx I just uploaded" — and Veta reads it directly.

Choosing the Right Model

Veta runs on Haiku by default — fast and great for most tasks. Switch to a more powerful model when the work demands it.

⚡ Default
Haiku
Fast everyday tasks, quick answers, simple drafts
/model anthropic/claude-haiku-4-5
🧠 Recommended
Sonnet
Complex documents, deep analysis, FAQ databases, Word & Excel work
/model anthropic/claude-sonnet-4-6
🚀 Most Powerful
Opus
Hardest reasoning tasks, strategic thinking, nuanced writing
/model anthropic/claude-opus-4-6
💡 Tip: Type the /model command at the start of your conversation when you know the task is complex. You can always switch back to Haiku with /model anthropic/claude-haiku-4-5 when done.

Working with Files

📤

Uploading

Visit vetalliance.website/files → enter password → drag & drop your file. Supported: PDF DOCX XLSX CSV TXT JPG/PNG — up to 50MB each.

💬

Referencing in chat

After uploading, just mention the filename to Veta: "Summarise meeting-notes.docx" or "Create a report from data.xlsx". Veta finds it automatically.

🗑️

Cleanup

Delete files you no longer need directly from the File Hub — just click the red trash icon next to any file.
